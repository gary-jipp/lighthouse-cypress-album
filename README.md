### Jest vs Cypress
* Jest
- CLI test runner
- based on assertions
- unit and integration testing
- FAST!

* Cypress
- Tests Run in its own browser
- Interacts with the app just like the user would
- visual
- Slow (relatively)

* install using npm (or yarn)
